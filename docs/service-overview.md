# ワタシのトリセツ — サービス説明書（SNS戦略用）

## 一言で言うと

**「友達と作る、自分の取扱説明書」**

自己診断（15問）＋友達からの回答（5問）で、自分では気づけない一面が見えてくる自己理解サービス。

---

## ターゲット

- 大学生（18〜22歳）
- 就活ガチ勢ではなく、友達関係・恋愛・SNSネタ・キャラ診断が好きな層
- 「自分ってどう見えてるんだろう？」が気になる人

---

## サービスの流れ（ユーザー体験）

### STEP 1：自分で15問に答える（約3分）

自分の性格について直感で回答。Big Five（外向性・協調性・開放性・誠実性・感受性）ベースの設問。

**設問例：**
- 「初対面の人とでも、割とすぐ打ち解けられる」
- 「友達の相談は、つい自分のことより優先してしまう」
- 「人の言葉が気になって、夜ふと考え込むことがある」

回答選択肢：めっちゃ当てはまる ／ まあまあ当てはまる ／ あんまり当てはまらない ／ ぜんぜん当てはまらない

### STEP 2：「仮トリセツ」が届く

診断直後に8タイプの中から自分のタイプが判明。トリセツ（取扱説明書）形式で結果が表示される。

ただし最初に見えるのは 8項目中3つだけ（基本スペック、喜ぶ言葉、エネルギーが上がる瞬間）。残りはロックされている。

### STEP 3：友達にLINEで送る

結果画面から友達招待リンクを送れる。LINEシェアボタン or リンクコピー。

**送られる側の体験：**
- ログイン不要
- 5問だけ・1分で完了
- 匿名で届く

### STEP 4：友達の回答でトリセツが育つ

| 回答人数 | 解放される内容 |
|---------|--------------|
| 0人 | 基本スペック、喜ぶ言葉、エネルギーが上がる瞬間 |
| 1人 | ＋苦手な環境 |
| 3人 | ＋取扱いのコツ、隠れ能力、気づいてない魅力、愛されるクセ（**完成版**） |
| 5人 | ＋自他ギャップ分析、総合分析（**深掘りレポート**） |

## 友達からの5問

友達に送られる質問は以下の5問：

1. **「この人は、知らない人がいる場でも自然に話せるタイプだと思う」**（4段階スケール）
2. **「この人は、困ってる人がいたら自分のこと後回しにしてでも助けるタイプだと思う」**（4段階スケール）
3. **「この人は、急な予定変更やハプニングも楽しめるタイプだと思う」**（4段階スケール）
4. **「この人の『ここが好き！』と思うところは？」**（選択肢：一緒にいると楽しい ／ 安心感がある ／ 刺激をもらえる ／ 素でいられる）
5. **「この人の、たぶん本人は気づいてない魅力は？」**（選択肢：実はめっちゃ繊細 ／ 実はめっちゃ頼りになる ／ 実はめっちゃ面白い ／ 実はめっちゃ優しい）

## 自他ギャップ分析（3人以上回答時）

自分の回答と友達の回答を比較して、以下3軸でギャップを可視化：

- 🗣️ **社交性** — 自分「ちょっと控えめ」 vs 友達「めっちゃ社交的」
- 🤝 **協調性** — 自分「まあまあ気配り上手」 vs 友達「かなり面倒見いい」
- 🌈 **好奇心** — 自分「安定志向」 vs 友達「割と柔軟」

ギャップが大きい軸には、ポジティブなフィードバック文が表示される。

**例：**
> あなたは自分の社交性を控えめに見ていますが、友達からは「もっと話せる人」と見られています。自分が思っているより、周りはあなたの存在に助けられています。

## バイラル設計（友達を呼ぶ仕組み）

### 友達招待の導線
- 結果画面から **LINEシェアボタン**（ワンタップ）
- リンクコピーボタン（LINEの他にも使える）
- 送信テンプレートが3パターンあり、自然な文体で送れる

**テンプレート例：**
> 自分の取説作れるやつやってみたんだけど
> 友達から見た自分も知りたくて
>
> 5問だけだから、気が向いたらやってみて〜

### 友達→診断の導線
- 友達が5問回答した後、完了画面で「自分のトリセツを作る」ボタンが表示される
- 回答した友達が自分も診断 → その友達がまた別の友達に送る、の連鎖を狙える
- 世代追跡あり（seed → gen1 → gen2 → ...）

### ロック解除によるモチベーション
- 最初は3/8項目しか見えない → 友達に送りたくなる
- 1人回答で4項目目が解放 → もっと集めたくなる
- 3人で全項目解放（完成） → 達成感 + 5人で深掘りレポートも...と続く
- 結果ページ再訪時も友達の声が更新されるため、何度も見に来る

## SNS戦略に使えるフック

### コンテンツの軸

| フック | 説明 |
|--------|------|
| **タイプ名のキャッチーさ** | 「暴走カリスマ」「沼ハマり探究者」など、自称したくなるネーミング |
| **自他ギャップ** | 「自分では控えめだと思ってたのに、友達からは社交的だって...」的な発見 |
| **友達の声** | 「ここが好き」「気づいてない魅力」は、もらうと嬉しい → スクショしたくなる |
| **愛されるクセ** | ネガティブな特徴もポジティブに変換 → 自虐ネタとしてシェアしやすい |
| **完成するプロセス** | 「3人集まって完成した！」というイベント性 |

### 相性のいいSNS

| SNS | 使い方 |
|-----|--------|
| **LINE** | メインの拡散チャネル。友達招待リンクの送付先 |
| **Instagram Stories** | タイプカードのスクショ + 「友達に聞いてみた」のストーリー |
| **X (Twitter)** | タイプ名 + 一言系の投稿、診断結果シェア |
| **TikTok** | 「友達に聞いたら意外な答えが...」系のリアクション動画 |
