# 32タイプ N軸差分コンテンツ（隔離ブランチ feature/32type-poc）

このディレクトリは、16タイプ × 繊細(N)/鉄壁(R) = 32パターンの文章コンテンツの
**バックアップ・レビュー用**生テキストです。

- キー形式: `ThirtyTwoTypeId`（`<base16>__N` / `<base16>__R`）
- N軸: `N`=繊細（感受性ゆたか・機微に気づく才能）/ `R`=鉄壁（動じない・揺るがない心強さ）
- 構成は本番16タイプと同一:
  - ① 自己診断本文 = 3セクション（取扱説明書 / 取扱注意ポイント / 相性の良いお相手）× 各2段落
  - ③ 強み・あれっ? = strengths 6項目 + surprises 6項目（{title, body}）
  - ④ 付き合い方コツ = 4〜5項目（{title, body}）
- `{B}` は友達名プレースホルダ。二人称は「アナタ」。
