2. コア体験フロー
─────────────────────────────────────────────

LP訪問 → 自己評価15問(3分) → 第1段階結果表示
→ 友達3人にLINE質問送信(1タップ) → 友達5問回答(1分)
→ 3人回答で完成版トリセツ→ さらに2人追加で深掘りレポート
→ SNSシェア → LINE登録

【段階アンロック構造】
- 1人:友達Aから見たワタシ(部分テキスト)
- 3人:他人から見たワタシ全体(スコア + テキスト)
- 5人:特別レポート(自他ギャップ深層分析)

【連続精度向上】
- 信頼度メーター
- タイプ判定確信度
- テキスト具体度の動的変化

※友達回答側はログイン不要・LINE登録不要・1分以内

3. 拡散の仕掛け
─────────────────────────────────────────────

   公式アカウントでは、診断結果例・シェア画像・使い方を発信。
   ユーザー投稿をUGCとして再活用する。
